# ベジタブルウォーズ
vegi-wars

## はじめに
- これはシューティングゲームです。
- Python スクリプトで書かれています。

## 動作環境
- Python 3.9
- ライブラリ
    - pygame 2.1


## 実行方法
- 以下のコマンドを入力します
```
python3 vegi.py
```
- または
```
python vegi.py
```

## 操作方法
- 移動
    - カーソルキー　右、左
- ミサイル発射
    - スペースキー

## 物語
地球は野菜たちの帝国に侵略されてしまった。プレイヤーは最後の希望として、勇敢な人参の戦士となり、野菜たちとの戦いに挑むことになる。
